iNatty NFTs Registration




----------------------------------------------------------------------------------

1. Get an iNaturalist.org Mobile App and Make an Account

Get iNaturalist account / mobile app and create an observations (e.g. take a picture of a house plant or a dog).


2. Copy This Token

3. Add Underscore and Token to your iNaturalist Username (e.g. chiggimps_[token])

For Authentication. On iNaturalist, go to Settings (upper right corner) >> Username


4. Enter iNaturalist User Name (e.g. chiggimps_kgtph021)

Wait a couple minutes before connecting wallet so the new username is registered



5. Select Observations to Mint as NFTs
